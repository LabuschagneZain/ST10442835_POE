CLDV6212_POE Web Application

This project is a web application built with ASP.NET Core MVC, Azure SQL, and Azure Blob Storage.
It includes user authentication, product browsing, cart management, order processing, and proof-of-payment uploads.

Features

User login and role-based access

Product display from Azure Blob Storage

Cart system using Azure SQL

Checkout and order creation saved to Azure Table Storage

File uploads (proof of payment) saved to Azure Blob Storage

Admin and customer dashboards

Technologies

ASP.NET Core MVC

Azure SQL Database

Azure Blob Storage

Azure Table Storage

Entity Framework Core
